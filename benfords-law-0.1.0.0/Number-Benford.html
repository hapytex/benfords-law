The Integral binary number that contains the binary start sequence (with an optional sequence of "leading zeros"), should be greater than zero.

The probability of a binary number starting with the givenn binary sequence. Unspecified if the number is less than zero.

Determine the probability of the sequence starting with the given digits of a binary number. Binary numbers always start with 1. If you thus call startSequence2' 9, you get the probability of a binary number starting with 1, 0, 0, and 1 as binary digits.
